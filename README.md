# IBM Bank Management System (C++)

A console-based Bank Management System built using C++ OOP concepts.

## Features
- Admin & Customer login
- Account creation & deletion
- Deposit, Withdraw, Transfer
- Transaction history
- Interest calculation
- File handling
- OOP (Classes, Encapsulation)
